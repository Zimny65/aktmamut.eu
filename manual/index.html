🧭 AKT MAMUT – Manual: EXPEDITIONS

📱 Krok 1 – Locus Map

  • Zarejestruj trasę
  • Edytuj trasę: Moja biblioteka → <TRAIL NAME>
  • Nazwij zgodnie ze standardem:
    2025-06-18 07:26 Nemira Mare
  • Aktualizuj wysokość:
    Narzędzia → Aktualizacja wysokości
  • Eksport GPX
  • Plik zapisany w:
Google Drive

🛠️ Krok 2 – GPX Track Editor

  • Otwórz plik GPX
  • Standaryzacja punktów:
    Filter → By value → Min distance = 10 m
  • Usunięcie nieścisłości:
    Filter → Local inconsistencies
  • Usuń zbędne punkty:
    Filter → Delete filtered points
  • Zapisz plik
Google Drive

🌍 Krok 3 – Wikiloc

  • Otwórz wikiloc.com
  • Zaloguj się
  • Załaduj plik GPX
  • Otwórz szczegóły trasy
  • Pobierz plik GPX (oryginalny)
OneDrive/AKT-MAMUT/data/gpx

📊 Krok 4 – Google Sheets

  • Otwórz arkusz AKT Mamut Expeditions
  • Dodaj nowy wiersz
  • Trail GPX
    link do trasy Wikiloc
  • Distance
    Długość śladu
  • Up
    Suma przewyższeń
  • Time
    Czas
  • GPX
    nazwa pliku z katalogu data/gpx

⚙️ Krok 5 – Generowanie mapy

Uruchom: 
generate_geojson.py
Program wykonuje: 
1. Pobranie Google Sheets
2. Backup CSV arkusza
3. Audyt GPX vs Sheets
4. Generowanie brakujących RAW GeoJSON
5. Scalanie do expeditions.geojson
6. Kopiowanie pliku do katalogu strony
Plik wynikowy: 
AKT-MAMUT/site/expeditions/expeditions.geojson

🧪 Krok 6 – Test lokalny

Terminal: 
cd OneDrive/AKT-MAMUT/site
Uruchom serwer: 
python -m http.server 8000
Otwórz: 
http://localhost:8000
Sprawdź czy nowa trasa jest widoczna.
